The width can be set as a specific size (in px, pt, cm, em, etc) or by using one of the three pre-defined values: thin, medium, or thick.

The color can be set by: name - specify a color name, like "red" ; HEX - specify a HEX value, like "#ff0000" ; RGB - specify a RGB value, like "rgb(255,0,0)" ; HSL - specify a HSL value, like "hsl(0, 100%, 50%)" ; transparent.
